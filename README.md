# VINA

### Virtual International News Assistant
#### Your personal news assistant.

VINA uses a Python server on a Raspberry Pi to deliver the current month's news on a glass sheet made in SwiftUI

#### The VINA project is split across 2 respositories: [VINA-Frontend](https://github.com/yosefahab/VINA-Frontend) and [VINA-Backend](https://github.com/yosefahab/VINA-Backend)
